Sebelum menggunakan aplikasi ini :
- Pastikan mendownload mediapipe, dan opencv-python
- Pastikan jalankan aplikasi ini di directory yang sama dengan modul-modul yang dibutukan (directkeys.py, dan HandTrackingModule.py)
- Memiliki kamera yang tidak sedang dipakai

--------------------------------------------------------------------------------------------------------------------------------------

Before using this application:
- Make sure to download mediapipe, and opencv-python
- Make sure to run this application in the same directory as the required modules (directkeys.py, and HandTrackingModule.py)
- Have a camera that is not being used
